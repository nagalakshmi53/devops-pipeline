# DevOps CI/CD Pipeline with Jenkins

This project demonstrates a basic Jenkins CI/CD pipeline as part of Task 2 for a DevOps internship.

## Files

- `index.html`: Simple static webpage
- `Jenkinsfile`: Contains the build, test, and deploy stages
- `README.md`: Explanation of project structure and purpose

## Instructions

1. Connect this repo to a Jenkins pipeline (CloudBees or local)
2. Ensure Jenkins is configured to pull the code from GitHub
3. Watch the pipeline stages run on each commit

## Outcome

Automated CI/CD process with build, test, and simulated deploy stages
